## Getting Started

1. Clone the repo

    ```bash
    git clone https://github.com/srjheam/ignite-platform.git
    ```

2. Head to the repo directory

    ```bash
    cd ignite-platform/
    ```

3. Install NPM packages

    ```bash
    npm install --force
    ```

4. Start development server and you're done

    ```bash
    npm run dev
    ```
